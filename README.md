# CurConverter
Веб-приложение для просмотра актуальных курсов валют ЦБ и выполнения конвертации.

# Функционал
1. Просмотр курсов валют:
- приложение отображает актуальные курсы валют относительно рубля;
- каждая валюта представлена кодом, названием, курсом и датой обновления;
2. Конвертация валют:
- пользователь может ввести сумму, выбрать исходную и целевую валюты, после чего получить результат;
- конвертация выполняется на основе актуальных курсов валют.
3. Автоматическое обновление данных:
- курсы валют автоматически загружаются из API ЦБ РФ и сохраняются в базе данных.

# Стек
C#, .NET, ASP.NET, Entity Framework Core, Microsoft SQL Server, XML API, Http, HTML, CSS
